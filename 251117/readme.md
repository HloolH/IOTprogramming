제1장 파이썬 소개 및 기본 학습 요약<br/>
이 문서는 프로그래밍의 기본 개념, 프로그래밍 언어로서의 파이썬 소개, 그리고 기본적인 개발 환경 설정 및 문법을 다루고 있습니다.

1. 컴퓨터와 프로그램<br/>
1.1 컴퓨터의 유연성
컴퓨터는 대단히 유연한 기계로, 소프트웨어만 바꾸면 리포트 작성부터 게임까지 다양한 작업을 수행할 수 있습니다.

1.2 프로그램의 정의<br/>
**프로그램(Program)**은 컴퓨터가 수행할 자세한 명령어(instruction)들의 리스트를 적어 놓은 문서입니다.

1.3 프로그래밍의 필요성<br/>
우리가 어떤 일을 하든 프로그래밍은 필수적인 기술이 되었으며 , 인문사회계 학생들도 파이썬을 이용하여 업무를 자동화하는 경우가 많습니다. 앞으로 단순 반복 작업은 컴퓨터가 맡게 될 것이며 , 인공지능, 로봇 프로세스 자동화(RPA) 등 다양한 분야에 활용됩니다.




2. 프로그래밍 언어와 파이썬<br/>
2.1 프로그래밍 언어<br/>
컴퓨터는 사람의 언어를 이해할 수 없기 때문에 프로그래밍 언어를 사용해야 합니다. 인간이 작성한 프로그램은 **컴파일러(또는 인터프리터)**라는 통역 소프트웨어를 통해 컴퓨터가 이해하는 기계어로 바뀝니다.


2.2 파이썬의 특징

개발자: 1991년 귀도 반 로섬(Guido van Rossum)이 개발한 대화형 프로그래밍 언어입니다.


생산성 및 효율성: 생산성이 뛰어나고, 간결하면서도 효율적인 프로그램을 빠르게 작성할 수 있습니다.


인터프리터 언어: 명령문의 결과를 즉시 볼 수 있어 초보 프로그래머에게 유리합니다.


라이브러리: 풍부한 라이브러리가 있으며 설치가 쉽습니다.

3. 개발 환경 및 실행 모드<br/>
3.1 개발 환경

아나콘다(Anaconda): 인기 있는 라이브러리가 거의 모두 포함된 파이썬 배포판입니다.


스파이더(Spyder): 파이썬으로 작성된 파이썬 개발 도구입니다.

3.2 실행 모드

대화형 모드(Interactive Mode): 콘솔(예: IPython 콘솔)에서 문장을 한 줄씩 입력하여 즉시 실행 결과를 확인하는 방식입니다.

스크립트 모드(Script Mode): 코드를 파일로 만들어서 저장한 후 파이썬 인터프리터가 이 파일을 한 번에 전부 읽어서 실행하는 방식입니다. 코드가 복잡해질 때 효율적입니다.

4. 파이썬 기본 문법<br/>
4.1 기본 계산
파이썬은 산술 연산자(+, -, *, /)를 사용하여 계산을 수행할 수 있습니다.

4.2 문자열 (String)

문자열은 큰따옴표("...")나 작은따옴표('...') 안에 들어 있는 텍스트 데이터입니다.


연결: + 연산자로 문자열을 연결할 수 있습니다 (예: ”나의 “ + “고양이” → 나의 고양이).


반복: * 연산자로 문자열을 반복할 수 있습니다 (예: "Hello" * 10).

4.3 print() 함수

print() 함수는 화면에 문자열이나 계산 결과를 출력할 때 사용되는 함수입니다. 여러 개의 값들을 차례대로 출력할 수 있습니다 (예: print("결과값은", 10, "입니다.")).
